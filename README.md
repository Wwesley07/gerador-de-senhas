# Gerador de senhas
Olá, tudo bem? Me chamo Wesley, venho aqui com um projeto de estudo, um gerador de senhas aleatórias com caracteres especiais. Utilizei a linguagem Python e alguns comandos básicos. Fica a disposição o projeto no GitHub, irei dar suporte de tempos em tempos. Espero que gostem.
## O que ele faz
Gera uma senha aleatória de 12 caracteres contendo letras maiúsculas, minúsculas, números e símbolos.
## Como rodar
1. Tenha o Python instalado
2. Abra o terminal na pasta do projeto
3. Execute o comando: python gerador.py
